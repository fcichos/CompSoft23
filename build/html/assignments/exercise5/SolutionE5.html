Lotka-Volterra Equation

Since we did already solve quiete some differential equations for mechanics problems during the lectures, we will use the Lotka-Volterra equations to do something new. These equations characterize the dynamics of biological systems in which a predator \(y\) population and a prey \(x\) population interact.

The prey population is described by

\begin{equation} \dot{x}=x(\alpha-\beta y) \end{equation}

where \(\alpha\) denotes the natural growth rate of the prey and \(\beta\) the death rate per encounter with the predator \(y\).

The

\begin{equation} \dot{y}=-y(\gamma-\sigma x) \end{equation}

where the predators die naturally with a rate \(\gamma\) and grow with a rate \(\sigma\) if there is enough prey \(x\).

Problem 1

Write a Python function that can be used with the odeint method of the scipy module.

[3]:

### BEGIN SOLUTION
def LV(state,t):
  x,y = state
  xd = x*(alpha - beta*y)
  yd = -y*(gamma - sigma*x)
  return([xd,yd])
### END SOLUTION

Define some initial parameters where all of the rates are 0.1. Define a time vector.
